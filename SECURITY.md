# Security Policy

## Supported Versions
This is a hobby/indie project. There is no formal security support window.

## Reporting
If you believe you found a security issue:
- Do **not** publish details immediately.
- Open an issue with minimal details, or contact the maintainers privately if contact info is provided in the project description.
